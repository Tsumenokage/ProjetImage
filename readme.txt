Notice : 

*Compilation du programme :
	Exécuter le MakeFile présent dans le dossier
	Si le MakeFile ne fonctionne pas à cause de librairie non trouvé, exéctuer la ligne de commande suivante : 
	"qmake-qt4 pro.pro"
	Executer le MakeFile une nouelle fois

*Exécution du programme : 
	Le progamme se compille sous le nom : *A REMPLIR*
	Lancer le programme avec la ligne de commande suivante : 
	"./NOM_PROGRAMME Chemin/Vers/Dossier/Image option"
	Le premier paramètre est le chemin vers le dossier contenant les images à traiter au format PNG
	Le second paramètre peut prendre deux valeurs :
		"b" Désactivera tous les affichaches d'images et affichera un benchmark du programme
		"s" Affichera tous les résultats otenues lors des différents traitements

Exemple : ./NOM_PROGRAMME ./images/log1 b
